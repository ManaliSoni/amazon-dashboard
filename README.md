# 📦 Amazon Store Sales Dashboard – Power BI

![Dashboard Preview](amazon_dashboard.png)

## Overview
A dark-themed Power BI dashboard analysing **Amazon Store sales data** across regions, product categories, customer segments, and time periods to uncover profitability trends.

## Key Metrics
| Metric | Value |
|--------|-------|
| Total Sales | $1.566M |
| Total Profit | $175K |
| Total Orders | 5,901 |
| Total Products | 1,742 |

## Key Insights
- **Consumer segment** accounts for 48% of sales, followed by Corporate (33%) and Home Office (19%)
- **Q4 is the strongest quarter** — sales spike significantly compared to Q1–Q3
- **Office Supplies** is the top-selling category ($644K), ahead of Technology ($471K) and Furniture ($452K)
- **Top 3 profitable sub-categories:** Copiers ($43K), Accessories ($25K), Phones ($22K)
- **Cash on Delivery (COD)** is the most preferred payment method at 43%
- **Standard shipping** dominates at 58% of all orders
- Technology has the **highest return rate** at 18%

## Visualisations
| Chart | Description |

| KPI Cards | Sales, profit, orders, products |

## Filters
- Region (Central, East, South, West)
- State

## Tools Used
- **Power BI Desktop** – data modelling, DAX, visualisations
- **Dataset** – Amazon Store Sales Dataset (2019–2020)

## How to View
1. Download `amazon_sales_dashboard.pbix`
2. Open in **Power BI Desktop**
3. Use region buttons and state filter to drill down by geography
